Fixing the Java security block

If you see this:

Try the fix below
  1. Open the Java Control Panel. Go to the Windows Start button, and search for Configure Java.
  2. Go to the Security tab.
  3. Click the Edit Site List... button.
  4. In the popped-up window Exception Site List Click Add, and type in the address of the Java applet.
  5. Click OK in the Exception Site List window.
  6. Click OK in the Java Control Panel window.
