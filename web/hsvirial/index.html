Notes and tips:
  • A red ball represents a failed Monte Carlo (MC) trial; a green ball represents a successful one.
  • The maximal dimension is 100, and the maximal order is 63.
  • Always hit “Enter” after changing any parameter to make sure it takes effect.
  • Click “MDS mode 0” to change the representations in the left and right panels. In modes 1 and 3, the left panel uses multidimensional scaling. In modes 2 and 3, the right panel uses multidimensional scaling. Multidimensional scaling is a way of squeezing high-dimensional objects in lower dimensions, and thus is useful in visualizing high-dimensional clusters here.
  • Click the right panel and scroll the mouse to change the scale there; drag the mouse to rotate the structure.
The source code of the above applet can be downloaded here VirSampApp.java. It was adapted from the C-based source code below. Although not fully-optimized, it offers a convenient to tool make rough estimates.

Complete source code

The complete C-based source code can be downloaded here vircode.zip. The package also contains several auxiliary Mathematica and Python scripts.
